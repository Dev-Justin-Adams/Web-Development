HTML Introduction

HTML (HyperText Markup Language) is the standard markup language used to structure content on the web. It forms the backbone of a webpage and uses elements, represented by tags, to define various parts of the content, such as headings, paragraphs, lists, and links.

HTML Basics

HTML consists of a series of elements. Each element is wrapped in angle brackets and typically consists of an opening tag, content, and a closing tag.

  • <h1>...</h1>: Defines a top-level heading.
  • <p>...</p>: Defines a paragraph.
  • <a href="url">...</a>: Creates a hyperlink.

HTML5 introduced new semantic elements such as <header>, <footer>, and <section>, which help in better structuring the content.

CSS Introduction

CSS (Cascading Style Sheets) is used to control the presentation of HTML elements. It allows you to change the appearance of your webpage, including layout, colors, fonts, and more, by linking styles to HTML elements.

Using CSS, you can control the design of multiple pages from a single external stylesheet or apply styles directly within HTML files.

CSS and HTML Together

HTML structures the content, while CSS is used to style and lay out that content. Let's look at an example where both are combined:

<!DOCTYPE html>
<html>
 <head>
  <link rel="stylesheet" href="styles.css">
 </head>
 <body>
  <h1>Hello, World!</h1>
  <p>This is a paragraph.</p>
 </body>
</html>

In this example, HTML defines the structure with headings and paragraphs, and the linked CSS (in "styles.css") will control how the page looks visually.

CSS Selectors

CSS selectors are used to target HTML elements that you want to style. Common selectors include:

  • Element selector: h1 { color: blue; } (Targets all <h1> elements)
  • Class selector: .button { background-color: red; } (Targets elements with the class="button" attribute)
  • ID selector: #header { font-size: 20px; } (Targets the element with the id="header" attribute)
CSS Box Model

Every HTML element is considered as a box, and the CSS Box Model is used to define the size and spacing of elements. It includes:

  • Content: The actual content of the box (e.g., text or images).
  • Padding: The space between the content and the border.
  • Border: The line surrounding the padding (or content if padding is not specified).
  • Margin: The space outside the border between other elements.

Example:

p {
  margin: 20px;
  padding: 10px;
  border: 1px solid black;
}
CSS Flexbox

Flexbox is a layout model that allows you to design flexible and responsive layout structures without using float or positioning.

Example:

.container {
  display: flex;
  justify-content: space-around;
}

This will align items inside the container evenly, with space between them.

CSS Grid

CSS Grid Layout is a two-dimensional layout system for the web that allows you to create complex layouts with rows and columns.

Example:

.grid-container {
  display: grid;
  grid-template-columns: auto auto auto;
  grid-gap: 10px;
}

This example creates a grid layout with three equal columns.
