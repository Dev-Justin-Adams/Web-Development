Introduction

JavaScript is a high-level, dynamic programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. It allows for dynamic and interactive content on web pages.

  • JavaScript is essential for creating interactive, responsive web applications.
  • It can be used to manipulate the DOM (Document Object Model), handle events, and create animations.
What you should already know

This guide assumes you have the following basic background:

  • Basic understanding of HTML and CSS.
  • Familiarity with how the web works and what browsers do.
  • Experience with a text editor to write JavaScript code.

Variables

Variables store data values in JavaScript. You can create variables using `var`, `let`, or `const`:

let name = "John";
const age = 30;

`let` allows you to reassign values, while `const` creates a variable that cannot be reassigned.

Data Types

JavaScript supports different data types, including:

  • String: A sequence of characters (e.g., `"Hello"`)
  • Number: Numeric values (e.g., `10`, `3.14`)
  • Boolean: True or False values (e.g., `true`, `false`)
  • Array: A collection of values (e.g., `[1, 2, 3]`)
  • Object: A collection of key-value pairs (e.g., `{name: 'John', age: 30}`)
Functions

Functions allow you to define reusable blocks of code. You can declare a function using the `function` keyword:

function greet() {
console.log('Hello!');
}

To call a function, simply use its name followed by parentheses:

greet();
Conditionals

Conditional statements control the flow of code based on conditions:

let age = 20;
if (age >= 18) {
console.log('You are an adult');
} else {
console.log('You are a minor');
}
Loops

Loops allow you to execute a block of code multiple times. The most common loop types are `for` and `while` loops:

for (let i = 0; i < 5; i++) {
console.log(i);
}
DOM Manipulation

JavaScript can manipulate HTML elements by accessing the DOM (Document Object Model). Use methods like `getElementById()` or `querySelector()` to select and manipulate elements:

document.getElementById('myElement').innerText = 'New Text';
Events

JavaScript allows you to handle user interactions through events. An event listener can be added to an element to trigger a function when a specific event occurs:

document.getElementById('myButton').addEventListener('click', function() {
alert('Button clicked!');
});
